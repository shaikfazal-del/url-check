HOW TO GET YOUR VIRUSTOTAL API KEY AND ADD IT TO url_check.py

1. Open any browser and go to this link:
   https://www.virustotal.com/gui/home/upload

2. Sign up for an account (or log in if you already have one).

3. After logging in, scroll down or go to this direct link to get your API key:
   https://www.virustotal.com/gui/my-apikey

4. Copy the API key shown on that page.

5. Open the file named url_check.py using any code editor (like VS Code, Notepad++, or even Notepad).

6. At the top of the file, find this line:
   API_KEY = ""

7. Paste your copied API key inside the quotes. For example:
   API_KEY = "your_actual_api_key_here"

8. Save the file.

You're all set to use the script with your VirusTotal API key!
